## Simple Chat Application using WebSockets


![alt text](demo.gif)

## Install Server and Client 

#### [ Install Server ] [ Open Terminal ] 
	
	$cd ./server_app 
	$./install.sh  

#### [ Install Client ] [ Open Terminal ] 
	
	$cd ./client_app 
	$./install.sh  



## Run Server [ Open Terminal 1 ] 
    
    $cd ./server_app 
	$./run.sh 



## Run Clients

#### Client 1 :  [ Open Terminal 2 ] 
    
    $cd ./client_app
	$./run.sh 

#### Client 2 :  [ Open Terminal 3 ] 
    
    $cd ./client_app
	$./run.sh
